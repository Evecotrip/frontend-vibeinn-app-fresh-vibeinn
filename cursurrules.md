# Vibeinn - Design & Development Guidelines

## App Vision

Vibeinn is an authentic college social app that connects students across universities worldwide, fostering genuine communities and meaningful interactions.

## Brand Identity

- **Name**: Vibeinn
- **Tagline**: "Connect. Share. Thrive."
- **Mission**: Creating authentic connections between college students globally

## Color Palette

- **Primary**: #0066FF - Used for CTAs, highlights, and interactive elements
- **Dark Background**: #121212 - For dark mode screens
- **Light Background**: #F5F5F5 - For light mode screens
- **Dark Text**: #000000 - For text on light backgrounds
- **Light Text**: #FFFFFF - For text on dark backgrounds

## App Structure

The app follows a four-tab navigation structure:

### 1. Home Tab

- Twitter-style feed with "For You" and "Following" top tabs
- Post types: text, images, and vertical videos
- Stories feature for ephemeral content
- Discover section for new connections and trending content

### 2. Events Tab

- Campus events calendar
- Event creation and RSVP functionality
- Ticketed events (future monetization after reaching 5-10M users)
- Event categories (academic, social, sports, etc.)

### 3. Community Tab

- Themed chat rooms (e.g., "Motivation Station")
- Vibe Circles for small, trusted groups
- Topic-based discussions
- Real-time support features

### 4. Profile Tab

- Life Chapters organization (e.g., "College Years")
- Achievements and badges display
- Privacy settings and preferences
- Content archive and memories

## Core Features

### Content Creation

- Text posts
- Image uploads
- Vertical video posts (TikTok/Reels style)
- Stories (24-hour ephemeral content)

### Community Building

- Real-time chat rooms
- Interest-based groups
- University-specific communities
- Cross-campus connections

### Life Stories & Milestones

- Chapter-based organization of memories
- Milestone celebration and recognition
- Achievement tracking
- Growth visualization

### AI-Powered Features

- Personalized content recommendations
- Mood-based music and activity suggestions
- Content moderation for safety
- Smart connection suggestions

### Privacy & Safety

- Customizable privacy controls
- AI-powered content moderation
- Reporting mechanisms
- Support resources

### Creative Expression

- Mood art sharing
- Playlist curation and sharing
- Creative challenges and prompts
- Multimedia expression options

### Gamification

- Mood streaks and check-ins
- Growth badges and achievements
- Community contribution recognition
- Challenges and rewards

### Integration Features

- Calendar syncing
- Wellness tools and resources
- Journaling capabilities
- Meditation and mindfulness features

## User Flow

### Onboarding

1. Sign-up/Login
2. Profile creation
3. Interest selection
4. University verification
5. Connection suggestions
6. Tutorial walkthrough

### Main Navigation

- Bottom tab bar with Home, Events, Community, and Profile
- Top bar with search, notifications, and messaging icons
- Contextual menus for additional features
